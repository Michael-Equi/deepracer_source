Use Case and High-Level Description

Text detector based on PixelNet architecture with MobileNetV2 as a backbone for indoor/outdoor scenes.

Inputs

  1. name: "input" , shape: [1x3x768x1280] - An input image in the format [BxCxHxW], where:

    • B - batch size
    • C - number of channels
    • H - image height
    • W - image width

Expected color order - RGB.

Outputs

  1. The net outputs two blobs. Refer to PixelNet for details.
    • [1x2x192x320] - logits related to text/no-text classification for each pixel.
    • [1x8x192x320] - logits related to linkage between pixels and their neighbors.
